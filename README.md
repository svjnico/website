# Gabby's Tree Service — Website Redesign

Production-ready single-page website for [gabbystreeservice.net](https://gabbystreeservice.net/).

## Open locally

Double-click `index.html` or open it in Chrome/Edge.

## What’s included

- SEO title/meta, Open Graph, LocalBusiness + FAQ schema
- Sticky nav, hero, trust bar, about, services (with modals)
- 24/7 emergency, HOA, seniors, “No Mess Left Behind”
- Before/after gallery, testimonials placeholders, blog placeholders
- Full FAQ (original copy preserved), areas served, contact form + Google Map
- Mobile sticky Call / Free Quote bar
- Tailwind CDN for easy customization

## Deploy

Upload `index.html` as the homepage on your host, or point a static host (Netlify, Cloudflare Pages, Vercel) at this folder.
